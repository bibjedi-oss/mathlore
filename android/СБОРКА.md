# Сборка Android-приложения Архи

## Перед первым открытием

1. Открой папку `android/` в **Android Studio** (File → Open → выбери эту папку)
2. Android Studio предложит синхронизировать Gradle — нажми **Sync Now**
3. Установи Android SDK если попросит (обычно уже есть)

## Добавление фоновых изображений в APK

Изображения нужно положить в:
```
android/app/src/main/assets/mathlore-assets/
```

Файлы (скачай из Supabase или из папки проекта):
- `bg-1.jpg` — фон 1 класс
- `bg-2.jpg` — фон 2 класс
- `bg-3.jpg` — фон 3 класс
- `bg-4.jpg` — фон 4 класс
- `bg-7.jpg` — фон 7 класс (Древняя Греция)
- `bg-8.jpg` — фон 8 класс (Золотой век ислама)
- `bg-9.jpg` — фон 9 класс (Эпоха Просвещения)

Приложение перехватит запросы к Supabase и отдаст эти файлы с устройства.
Если файла нет в assets — запрос уйдёт в сеть как обычно (fallback).

## Изменить адрес сервера

В `MainActivity.kt`, строка `SERVER_URL`:
```kotlin
private val SERVER_URL = "https://mathlore.ru"
```
Замени на актуальный URL продакшн-сервера.

## Сборка APK для тестирования

В Android Studio: **Build → Build Bundle(s) / APK(s) → Build APK(s)**

Готовый файл появится в:
```
android/app/build/outputs/apk/debug/app-debug.apk
```

## Сборка для RuStore (release)

1. **Build → Generate Signed Bundle / APK**
2. Выбери **APK**, создай keystore (или используй существующий)
3. Выбери конфигурацию **release**
4. Готовый APK загружай в RuStore

## Иконки приложения

Замени заглушки в папке `res/mipmap-*/`:
- `ic_launcher.png` — стандартная иконка
- `ic_launcher_round.png` — круглая иконка

Размеры: mdpi 48×48, hdpi 72×72, xhdpi 96×96, xxhdpi 144×144, xxxhdpi 192×192.

Или используй **Image Asset Studio** в Android Studio:
File → New → Image Asset → Launcher Icons
